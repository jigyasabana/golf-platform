# Golf Platform 

A full-stack golf engagement platform built using React and Supabase.

## Features
- User Authentication (Signup/Login/Logout)
- Score Tracking (stores last 5 scores only)
- Charity Selection
- Subscription System (Monthly/Yearly)
- Random Draw System with match logic

## 🛠 Tech Stack
- React (Vite)
- Supabase
